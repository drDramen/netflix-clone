## Run Locally

Clone the project

```bash
  git clone https://github.com/drDramen/netflix-clone.git
```

Go to the project directory

```bash
  cd netflix-clone
```

Install dependencies

```bash
  npm i
```

Start the server

```bash
  npm run dev
```

## Build

To build this project run

```bash
  npm run build
```
